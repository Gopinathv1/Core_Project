# Enterprise Manufacturing AI

This project contains the end-to-end workflow for manufacturing defect analysis and AI modeling.

## Structure
- data/raw: raw input data
- data/processed: cleaned/processed datasets
- notebooks: experiments and notebooks
- src/data: data loading and preparation modules
- src/eda: exploratory analysis scripts
- src/statistics: statistical analysis scripts
- src/features: feature engineering scripts
- src/ml: machine learning pipelines
- src/dl: deep learning modules
- src/autoencoder: anomaly detection models
- src/explainability: model interpretation tools
- src/api: deployment endpoints
- src/utils: reusable utility functions
